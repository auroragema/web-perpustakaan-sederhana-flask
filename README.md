# web-perpustakaan-sederhana-flask
Simple library management system using Flask and Python

<b> Aplikasi Web Manajemen Perpustakaan (Flask)</b>
Proyek ini merupakan aplikasi web manajemen perpustakaan sederhana yang dikembangkan menggunakan Python dan framework Flask. Aplikasi ini menyediakan fitur untuk menampilkan daftar buku, mencari buku, melihat detail buku, serta mengajukan peminjaman buku melalui antarmuka web. Proyek ini dirancang sebagai project pengembangan aplikasi web untuk menerapkan konsep dasar routing Flask, template HTML, pemisahan frontend–backend, serta pengelolaan data berbasis file CSV.

<b> Fitur Aplikasi </b>
1. Halaman Beranda (Home)
2. Halaman Daftar Buku
3. Halaman Detail Buku
4. Fitur Pencarian Buku
5. Form Peminjaman Buku
6. Halaman Konfirmasi Peminjaman
